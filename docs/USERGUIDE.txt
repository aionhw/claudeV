# claudeV User Guide

`claudeV` is a native X11 waveform viewer for VCD files.

## 1. Prerequisites

- Linux/Unix desktop with an X11 display server (or XWayland).
- Rust toolchain (`cargo`, `rustc`) installed.

## 2. Build

```bash
cargo build
```

For an optimized build:

```bash
cargo build --release
```

## 3. Run

Run with no file:

```bash
cargo run
```

Run with a VCD file:

```bash
cargo run -- path/to/file.vcd
```

Run with an explicit display:

```bash
cargo run -- -d :0 path/to/file.vcd
```

Load built binary directly:

```bash
./target/debug/claudeV path/to/file.vcd
```

## 4. UI Overview

The window has three main work areas:

- Module Browser (top-left): scope tree.
- Signals (bottom-left): signals in the selected scope.
- Wave (right): pinned waveform rows.

There is also:

- Top menu bar (`File`, `Edit`, `Search`, etc.; only `File` has actions).
- Toolbar (filename, zoom, view range, focus, filter).
- Overview strip (global time-range context).
- Marker dock (A/B markers and cursor readouts).
- Status bar (current focus and action status).

## 5. Quick Workflow

1. Open a VCD file from CLI or `File -> Open`.
2. In Module Browser, pick a scope (`j/k`, arrows, or mouse click).
3. In Signals focus, add desired signals with `a` or `Enter`.
4. Switch to Wave focus with `Tab` and inspect/reorder wave rows.
5. Zoom/pan/cursor/marker controls to measure time deltas.

## 6. Keyboard Controls

### Global

- `q` / `Esc`: quit.
- `s`: load built-in sample VCD.
- `Tab`: cycle focus `Browser -> Signals -> Wave`.
- `/`: start filter editing.
- `X`: clear filter.
- `+` or `=`: zoom in.
- `-` or `_`: zoom out.
- `f` or `0`: fit full timeline.
- `h` / Left Arrow: pan left.
- `l` / Right Arrow: pan right.
- `H`: jump to start.
- `L`: jump to end.
- `c`: set cursor to center of current view.
- `C`: clear cursor.
- `m`: set active marker (A or B) at cursor (or view center if cursor is off).
- `M`: toggle active marker (A/B).
- `D`: clear markers.
- `[` / `]`: nudge cursor left/right.
- `n` / `N`: jump cursor to next/previous edge on selected wave signal.
- `?`: show compact help text in status bar.

### Module Browser Focus

- `j` / Down Arrow: next scope.
- `k` / Up Arrow: previous scope.
- `PageDown` / `PageUp`: page navigation.
- `Enter` / `e`: expand/collapse selected scope.
- `A`: add all signals in selected scope recursively.

### Signals Focus

- `j` / Down Arrow: next signal.
- `k` / Up Arrow: previous signal.
- `PageDown` / `PageUp`: page navigation.
- `a` / `Enter`: add/remove selected signal from Wave.
- `A`: add all signals in selected scope.

### Wave Focus

- `j` / Down Arrow: next wave row.
- `k` / Up Arrow: previous wave row.
- `PageDown` / `PageUp`: page navigation.
- `d` / `Delete` / `Backspace`: remove selected signal row (or collapse bit-slice row).
- `J`: move selected signal down.
- `K`: move selected signal up.
- `e` / `E` / `Enter`: expand/collapse bus rows into bit slices.

### File Browser Mode (`File -> Open`)

- `Up` / `Down` or `k` / `j`: move selection.
- `PageUp` / `PageDown`: page navigation.
- `Enter` / `e`: open selected directory or file.
- `Esc`: cancel open dialog.

## 7. Mouse Controls

- Click top `File` menu:
  - `Open`: file browser panel (shows directories and `.vcd` files).
  - `Reload`: reload current file.
  - `Exit`: quit.
- Click module row to select scope.
- Click signal row to select signal.
- Drag signal row from Signals panel to Wave panel to pin it.
- Click wave row to select it.
- Click ruler/wave area to place cursor at clicked time.
- Drag splitters:
  - Vertical splitter between left and right panels.
  - Vertical splitter between signal-name/value and wave area.
  - Horizontal splitter between Module Browser and Signals panels.
- Mouse wheel:
  - Over Wave area: zoom in/out around pointer.
  - Over Signals area: scroll vertical list.
  - `Shift` + wheel (or buttons 6/7): horizontal scroll in Signals list.

## 8. Scopes, Filtering, and “All Scopes”

- The Module Browser includes a synthetic `(all scopes)` row.
- Selecting `(all scopes)` makes the Signals list global.
- Filter (`/`) matches `signal name` or `full hierarchical name`.
- Filter is case-sensitive.

## 9. Troubleshooting

- No window appears / display error:
  - Ensure `DISPLAY` is set and X11 is accessible.
  - If on Wayland, make sure XWayland is available.
- `File error` when loading:
  - Confirm path exists and is readable.
  - Use `File -> Open` to browse and select.
- `Parse error`:
  - Input may not match supported VCD constructs.
  - Try with a simpler/known-good VCD file.
- `Reload unavailable: no file loaded`:
  - Load a file first (CLI path or `File -> Open`).
